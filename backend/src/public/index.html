Bienvenidos a Starpi

En esta mini API vamos a poder obtener datos de StarWars más o menos lo que haya podido agregar al momento en que hayas encontrado esto jajajaj


Es importante que tengas una carpeta Assets para las imagenes en tu proyecto ya que tendrás que descargarlas y la api devolverá una ruta por defecto de dicha imagen

Como por ejemplo seria algo así: './assets/img/Imagen, así que para evitar errores agrega en el src de tu app una carpeta de assets.
