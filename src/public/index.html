Bienvenidos a Starpi

En esta mini API vamos a poder obtener datos de StarWars más o menos lo que haya podido agregar al momento en que hayas encontrado esto jajaja


Es importante que tengas una carpeta Assets para las imagenes en tu proyecto ya que tendrás que descargarlas y la api devolverá una ruta por defecto de dicha imagen

Como por ejemplo seria algo así: './assets/img/Imagen, así que para evitar errores agrega en el src de tu app una carpeta de assets.

Pulsa para descargar imagenes en zip

Como usar Starpi ?

Es realmente sencillo, solo tienes que colocar alguno de los siguientes Endpoints

Nota: Todos los Endpoints devuelven un archivo JSON y algunos facilitan la paginación de los mismos

Leyenda: Funciona// No Funciona
  1. /starpi

    Para consumir la api en su totalidad desde solo una llamada

  2. /starpi/starships

    Para consumir paginadamente la lista de naves

  3. /starpi/people

    Para consumir a personajes

  4. /starpi/planets

    Para consumir la lista de planetas
